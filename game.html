Game Summary

  • Genre: Government simulation / city management / light narrative adventure
  • Platform: HTML5 / Web (Desktop)
  • Engine: Custom web-based loop (HTML/JS/CSS)
  • Core Loop: You are Mayor Octavius of a seaside town. 1 game hour = 1 real minute; tasks arrive hourly during a 9–5 workday. Citizens follow daily routines and submit requests via the mailbox system.
