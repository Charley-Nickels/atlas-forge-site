Scenario selector

Switch between an RPG crafting patch, a KPI spreadsheet, a safety policy, a pizza delivery app, a restaurant site, and a media clipper.

Tabs with intent

Brainstorm toggles control the rough idea; UAT lists test steps; Review summarizes risks; Patch brings up a fake loading overlay.

Fake, but honest

Nothing touches your real files. The prototype is deliberately “obviously fake” while still revealing how an Atlas‑V flow could feel.

What AtlasStudio is (and isn’t)

This is a visual explanation tool, not a SaaS product. You can click through flows, flip toggles, run fake patches, and imagine how the underlying engine would behave. If you’re seeing this on GitHub or Vercel, it’s meant as a portfolio piece and a conversation starter.
